Kuka Sunrise project to implement ROS 2 Humble control of KMR IIWA 14 R820.  This is the Java only workspace for installing on the robot that acts as the driver.  It was tested in Kuka Sunrise workbench 1.16.  You will need the KMRIIWA_ROS2_WS repo for the ROS side of this.

This is a work in progress and I will update the readme when I have it stable.  Current state is I am able to jog the KMR robot with the keyboard.
